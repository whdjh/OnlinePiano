## 프로젝트 소개
사이드 토이 프로젝트로 혼자 진행한 웹으로 피아노를 칠 수 있는 웹페이지 입니다.

개발 배경에는 여행시나 어디서든 악기를 연습하고싶은데 악기를 가져갈 환경이 안되거나

장소에 구애를 받지않고 악기를 연습할 수 있는 방법으로 웹페이지를 이용한 악기 웹페이지가 있으면 좋겠다고 생각해서 개발하게 되었습니다.
